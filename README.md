# Kidz

Kidz — рабочее название AI-ассистента по детскому гардеробу. Продукт помогает родителю за 30 секунд решить, что надеть ребёнку сегодня, поддерживает актуальное состояние вещей и заранее предупреждает о пробелах в гардеробе.

Первый релиз ориентирован на родителей детей 2–10 лет. Детский режим, задания, звёзды, вишлисты и покупки входят в стратегию, но не в MVP: сначала нужно доказать ежедневную полезность основного сценария и выстроить контур защиты детских данных.
